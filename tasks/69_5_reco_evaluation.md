# Task 69.5: Reco Evaluation + Metrics

## Goal
Measure and validate recommendation quality.

## Scope
- Offline metrics: Recall@K, NDCG@K, coverage, cold-start coverage.
- Online metrics hooks: CTR, add-to-cart, conversion lift, basket value.
- Segmentation by location/channel + DemandKey family.
- Backtest dataset generation.

## Acceptance Criteria
- Evaluation pipeline runs on sample data.
- Metrics reported with pipeline/model version lineage.
