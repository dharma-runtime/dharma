# Workspace Philosophy (Planned)

The Workspace is the future GUI for DHARMA. It is designed to be **local-first**, **auditable**, and **DHL-driven**.

## Core Ideas

- **Buffers for Truth:** A buffer can be text, a subject state, or a query.
- **Views from Law:** DHL `view` blocks define UI without hard-coding screens.
- **Audit by Default:** Every UI element can explain the assertion that produced it.
